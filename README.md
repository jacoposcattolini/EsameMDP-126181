# 📌 BeatFighter

BeatFighter è un RPG rhythm game in stile cyberpunk. Il giocatore sceglie se giocare una modalità Storia (con storytelling e ordine dei boss prefissato) o una modalità Arcade (non si può morire e non ci sono limiti sulla scelta del boss che si vuole combattere). Sono presenti tre personaggi, chiamati Star(ognuno con un'abilità unica) utilizzabili per affrontare i boss in un combattimento stile Guitar Hero: colpendo le note, a tempo con la base musicale, premendo i tasti A / S / K / L si infligge danno al boss, sbagliando si subisce danno. Le Star si possono potenziare con xp guadagnata durante i combattimenti.

---

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle

### Istruzioni

```bash
git clone <https://github.com/jacoposcattolini/EsameMDP-126181>
cd <BeatFighter>
```

### Build del progetto
```bash
./gradlew build
```

### Esecuzione
```bash
./gradlew run
```
